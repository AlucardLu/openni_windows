========================================================================
    CONSOLE APPLICATION : Ni9PixelTest Project Overview
========================================================================

Options for command line:
	-r [RESOLUTION]
	   RESOLUTION: VGA, vga, QVGA, qvga.

	-f [RATIO]
	   RATIO: A floating number between 0 and 1.

	-K [RECORDING_FRAME]
	   RESORDING_FRAME: The amount of frames to be recorded. Should be >= 10.

	-b [BLOCK_SIZE]
	   BLOCK_SIZE: The size of the block to grab distance. Unit: pixel.
	   eg. If BLOCK_SIZE = 3, a 3*3 block will be used to capture distance at each point.
